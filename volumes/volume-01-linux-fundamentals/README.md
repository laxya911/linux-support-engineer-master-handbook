# Volume 1: Linux Fundamentals

This volume serves as the foundational corner of the **Linux Support Engineer Master Handbook**. It is designed to take you from a terminal novice (or someone coming from a GUI-dominated operating system like Windows Server) to a confident, CLI-capable support engineer.

---

## Volume Details

* **Estimated Reading & Lab Time**: 120 Hours
* **Difficulty Level**: Beginner → Junior Support Engineer
* **Total Chapters**: 30
* **Core Topics**: Shell navigation, text filtering, user administration, processes, packages, storage mounting, basic networking, and basic scripting.

---

## Target Objectives

By the end of this volume, you will be able to:
1. Navigating headless servers confidently using the command-line interface.
2. Locate, read, edit, and search for configuration files and log outputs.
3. Manage local user accounts, passwords, and file permissions (including standard Unix permissions and POSIX access control lists).
4. Inspect running processes, manage system services using `systemd`, and trace boot workflows.
5. Perform software maintenance using distribution package managers (`apt`, `dnf`, `yum`).
6. Partition disks, construct file systems, mount storage, and manage Logical Volume Manager (LVM) volumes.
7. Configure static and dynamic IP networking, diagnose connection errors, and remotely manage hosts over SSH.
8. Automate daily administrative checkups with cron jobs and write simple bash scripts.
